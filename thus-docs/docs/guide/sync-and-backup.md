## 备份到外部

备份到本地还不够？数据流转起来才是重点。

我们还支持 [实时备份](./connect/index) 到 WPS / 钉钉 / 维格表，在他们的多维表格上查看你的记事，给你一种批奏折的即视感。

我们真的要让你的数据，由你掌握！
